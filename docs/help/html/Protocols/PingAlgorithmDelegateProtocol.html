icmpPacketReceivedWithData:identifier:sequenceNumber:roundTripTime:type:from:

Informs about the reception of an ICMP packet.

- (void)icmpPacketReceivedWithData:(NSData *)packet identifier:(int)identifier sequenceNumber:(int)seqnum roundTripTime:(NSTimeInterval)dt type:(PingAlgorithmICMPType)type from:(NSString *)src

Parameters

packet

The IP address of the target network location.

identifier

The ping operation identifier.

seqnum

The packet sequence number.

dt

The packet round-trip time.

type

The type of packet.

src

The IP address of packet source (it can be equal to target).

Discussion

This method informs the ping algorithm delegate that an ICMP reply packet has been received in response to the packets transmitted using the method performWithTarget:numberOfPackets:maxNumberOfHops:packetSizeInBytes:maxNumberOfTries: .

pingPerformedWithTarget:identifier:numberOfPackets:maxNumberOfHops:packetSizeInBytes:

Informs about the successful transmission of ICMP Echo packages.

- (void)pingPerformedWithTarget:(NSString *)target identifier:(int)identifier numberOfPackets:(int)npackets maxNumberOfHops:(int)hops packetSizeInBytes:(int)size

Parameters

target

The IP address of the target network location.

identifier

The ping operation identifier.

npackets

The number of ICMP Echo packets to be send.

hops

The maximum number of times, a packet can be forwarded by an intermediate network location (value of the IP TTL field).

size

The size of ICMP Echo packet.

Discussion

This method informs the Ping algorithm delegate that the method performWithTarget:numberOfPackets:maxNumberOfHops:packetSizeInBytes:maxNumberOfTries: has successfully transmitted the ICMP Echo packets.
